Atividade 3

Seção 1

CSS (Cascading Style Sheets) é uma linguagem usada para definir o estilo de elementos HTML em páginas da web. Ele permite controlar cores, fontes, tamanhos e layouts de maneira eficiente. Com o CSS, é possível deixar o site visualmente atraente e responsivo. Ele separa o conteúdo (HTML) da apresentação visual. Isso facilita a manutenção e reutilização de estilos.

Seção 1.1

O CSS pode ser aplicado de forma inline interna ou externa. O estilo inline é adicionado diretamente no elemento HTML com o atributo style. O estilo interno é escrito dentro da tag style no cabeçalho da página. Já o CSS externo é feito em um arquivo separado com extensão css e vinculado ao HTML. A forma externa é a mais recomendada.

Seção 2

Seletores são usados para escolher quais elementos HTML serão estilizados. O seletor de tipo aplica estilos a todas as tags daquele tipo como parágrafos ou títulos. O seletor de classe pode ser reutilizado em vários elementos. O seletor de ID deve ser único para cada elemento. Também existem seletores combinados e avançados para maior controle.

Seção 2.1

CSS melhora a organização do código e permite reutilização de estilos. Com ele a aparência do site pode ser modificada sem alterar o HTML. Ele também permite criar designs responsivos que funcionam em diferentes tamanhos de tela. O CSS acelera o carregamento do site ao reduzir repetições de código. E ainda facilita a manutenção e colaboração entre desenvolvedores.
